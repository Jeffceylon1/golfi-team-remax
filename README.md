# Golfi Real Estate — Website

Hamilton & Niagara real estate website for the Golfi Real Estate.

Built on the HouseBox HTML template with Golfi Real Estate branding.

## Stack
- Static HTML / Bootstrap / jQuery
- Hosted on Vercel (canadamart team)
- AI Agent layer: **in progress** (Claude API + Supabase + Twilio)

## Pages
- `index.html` — Homepage
- `dashboard.html` — Agent Dashboard
- `sidebar-grid.html` — Property Listings
- `property-details-v*.html` — Property Detail Pages
- `about-us.html`, `our-service.html`, `contact.html`, `faq.html`
- `blog.html`, `blog-grid.html`, `blog-detail.html`
- `loan-caculator.html`, `pricing.html`

## AI Agent (Phase 1 — coming)
- Claude-powered chat widget with Hamilton/Niagara knowledge
- Passive lead capture (property saves, valuations, exit intent)
- WhatsApp + email alerts via Twilio + Resend
- Lead CRM dashboard in Supabase
